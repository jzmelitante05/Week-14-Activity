# 2. Installation Steps

Follow the steps below to install the application.

---

## Step 1 — Download the Installer

Download the installer file:

```plaintext
Setup.exe
```

Save it to your computer.

---

## Step 2 — Run the Installer

1. Locate the downloaded installer.
2. Right-click the file.
3. Select:

```plaintext
Run as Administrator
```

---

## Step 3 — Start Setup Wizard

The setup wizard will appear.

Click:

```plaintext
Next
```

to continue.

---

## Step 4 — Accept License Agreement

Read the license agreement carefully.

Select:

```plaintext
I Accept the Agreement
```

Then click:

```plaintext
Next
```

---

## Step 5 — Choose Installation Folder

Select the installation directory.

Example:

```plaintext
C:\Program Files\InventorySystem\
```

Click:

```plaintext
Next
```

---

## Step 6 — Install the Application

Click:

```plaintext
Install
```

Wait for the installation process to finish.

---

## Step 7 — Complete Installation

Click:

```plaintext
Finish
```

to complete the installation.

---

# 3. First-Time Setup

---

## Step 1 — Launch the Application

## Step 2 — Create Administrator Account

Click:

```plaintext
Create Account
```

## Step 3 — Configure Preferences

Adjust application settings:

- Theme

- User Preferences

---

## Step 4 — Complete Setup Tutorial

The system may provide a short tutorial for first-time users.

Follow the instructions displayed on the screen.

# 4. Troubleshooting

This section provides solutions to common installation problems.

---

## Problem 1 — Installer Does Not Open

### Solution

- Right-click installer
- Select:

```plaintext
Run as Administrator
```

---

## Problem 2 — Antivirus Blocks Installation

### Solution

Add the installer to antivirus exclusions.

---

## Problem 3 — Missing .NET Framework

### Error Example

```plaintext
.NET Framework 4.8 is required.
```

### Solution

Install .NET Framework 4.8 from Microsoft.

---

## Problem 4 — Application Crashes on Startup

### Solution

- Restart the computer
- Reinstall the application
- Check system requirements

---

## Problem 5 — Missing Files Error

### Solution

Reinstall the application completely.

# Frequently Asked Questions (FAQ)

---

## Q1: Is internet connection required?

No. The application can run offline.

---

## Q2: Can the installation folder be changed?

Yes. During setup, choose a different folder location.

---

## Q3: How do I uninstall the application?

Go to:

```plaintext
Control Panel → Programs and Features
```

Select the application and click:

```plaintext
Uninstall
```

---

## Q4: What should I do if I forget the administrator password?

Contact the system administrator or support team.
